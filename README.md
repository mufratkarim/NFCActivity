# NFCActivity
An NFC Starter App in Kotlin that is used to read various tags(NDEF, RFID, FeliCa, ISO 14443, etc) in Kotlin/Android

# Features
- Modern Gradle Builds
- Kotlin Android

# Documentation
[Full Guide](https://developer.android.com/develop/connectivity/nfc/nfc)
